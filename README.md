# CAN-module---EV-GoKart
Two-node CAN network built for an EV gokart for SAE with MCP2515 controllers.

made for 'FORMULA KART RACING CHAMPIONSHIP SEASON 06'
A complete CAN bus telemetry system for a competitive electric go-kart. Reads motor data like RPM, temperature, and voltage from the VESC motor controller over CAN, and displays real-time metrics on an onboard LCD during practice and race runs.
